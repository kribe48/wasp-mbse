In this folder you find:

- The code frame for your task. Use a fresh installation of Eclipse Neon (Modelling tools) and install the Xtext plugin. You should also have Ros and the Ros simulator running on your machine, so that you can test whether you create the correct python code.

- example.TurtleBotMission.DSL, example2.TurtleBotMission.DSL: two example turtle-bot mission files that your DSL should be able to open and consume for code generation

- ros_node: an example python file that could be the result of your code generation
